# DistributedAlgorithms

Information and code in this repository is used for university coursework

## LAB1 - HS Algorithm
In order to test, head to Lab1/Testing/Release/net8.0 folder and launch Launcher.exe.

## LAB2 - Maximum Independent Set
Arguments for console command are numbers from 1 to 6 that account for a graph file located in the upper directories

## LAB4 - Dining Philosophers
Arguments for console are either "single" for shared semaphore implementation or any other for mutex + individual semaphores.

## LAB5 - Locally Finite Algorithms
Both programs work with hardcoded graphs inside of graph_parse func, graphs should have .canvas extension.

## LAB6 - Atomic Swap
Lab implements a hard coded swap, where process A has 10 BTC and wants 50 ETH, process B has 1000 COINS and wants 10 BTC, process C has 50 ETH and wants 1000 COINS. A is a starter process.
